Project Description:
An interactive rule-based chatbot built in Python that responds to user inputs using predefined keywords and dictionary mapping.


Features:
•Keyword-based input detection
•Dictionary-based response system
•Colored console UI
•Typing delay simulation
•Exit functionality


Technologies Used:
•Python
•Dictionaries
•String processing
•Time module


How to Run:
python chatbot.py


Learning Outcomes:
•Modular programming
•User interaction logic
•String normalization and matching
